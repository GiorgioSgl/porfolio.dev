A simple [portfolio](https://giorgiosegallaportfolio.netlify.app/) about me.

## How to start

1. Install the required modules:

```bash
npm install
```

2. Run the porject:

```bash
npm run dev
```

npm run dev

## Deploy on netlify

When you push to the main branch, the project will be automatically deployed on Netlify.
